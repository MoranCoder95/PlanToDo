# PlanToDo

PlanToDo=任务代办+番茄钟+...

:sparkles:一个使用 electron + vue 3.0 + vite + ElementPlus开发的桌面应用

## Features

📦 任务新建修改完成

🎯 定时番茄钟提醒

🌱 定时锁屏锁屏



## 交流群

如何在有限的时间和精力下，以更高效、更健康的方式完成各项任务？是我们一直在实践探索的问题。欢迎加入交流群
